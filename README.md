# data-analysis

Personal data analysis projects by Dário — exploring real-world datasets with Python and pandas.

## Projects

### FIFA 19 — Player Analysis
Exploratory analysis of the FIFA 19 dataset (10,000+ players).

| Notebook | Description |
|---|---|
| `01_exploracao.ipynb` | Dataset structure, null values, basic statistics |
| `02_limpeza.ipynb` | Data cleaning and type conversion |
| `03_filtros.ipynb` | Filtering and selecting players |
| `04_groupby.ipynb` | Aggregations by position, club and nationality |
| `05_colunas_derivadas.ipynb` | Creating new columns and categories |
| `06_posicoes.ipynb` | Comparing attributes by position |
| `07_clubes.ipynb` | Club rankings by value and wage |
| `08_jovens.ipynb` | Most promising players under 23 |
| `09_time_ideal.ipynb` | Building a team within a budget |
| `10_relatorio.ipynb` | Final report and conclusions |

## Stack

- Python 3
- pandas
- Jupyter Notebook

## Dataset

Download the FIFA 19 dataset on [Kaggle](https://www.kaggle.com/datasets/karangadiya/fifa19).
